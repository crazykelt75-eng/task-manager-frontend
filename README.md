## Description

A task manager web app built with React. Users can register an account, log in, and then add, view, edit, delete, and mark tasks as complete. Each user only sees their own tasks.

---

## Technologies Used

- React
- React Router
- Bootstrap 5
- Vite
- JavaScript (ES6+)

---

## Pages / Components

- `Navbar` – navigation bar, shows different links depending on login state
- `Home` – landing page with app description and links
- `List` – shows all tasks for the logged in user
- `Details` – shows full info for one task, allows editing and deleting
- `AddTask` – form to create a new task
- `Login` – login form
- `Register` – register a new account

---

## How to Run Locally

1. Clone the repository:
```
git clone https://github.com/crazykelt75-eng/task-manager-frontend.git
```

2. Go into the folder:
```
cd task-manager-frontend
```

3. Install dependencies:
```
npm install
```

4. Start the development server:
```
npm run dev
```

5. Open your browser and go to `http://localhost:5173`

> Note: The app connects to the live backend so no extra setup is needed for that.
